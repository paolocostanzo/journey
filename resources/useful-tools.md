## üîß Essential Tools

### Reconnaissance
- **Nmap** - Network scanner
  ```bash
  nmap -sC -sV -oN scan.txt [TARGET]
  ```
- **Gobuster** - Directory/file brute-forcing
- **ffuf** - Fast web fuzzer
- **WhatWeb** - Website fingerprinting

### Web Application Testing
- **Burp Suite Community** ‚≠ê HTTP proxy and testing suite
- **OWASP ZAP** - Free alternative to Burp
- **SQLMap** - Automated SQL injection tool
- **Nikto** - Web server scanner

### Exploitation
- **Metasploit Framework** - Exploitation toolkit
- **Exploit-DB** - Exploit database
- **SearchSploit** - Local exploit search

### Password Attacks
- **Hydra** - Network login brute-forcer
- **John the Ripper** - Password cracker
- **Hashcat** - Advanced password recovery

### Post-Exploitation
- **LinPEAS** - Linux privilege escalation scripts
- **WinPEAS** - Windows privilege escalation scripts
- **GTFOBins** - Unix binaries privilege escalation database

### Scripting & Automation
- **Python** ‚≠ê Primary scripting language
- **Bash** - Shell scripting for automation
- **PowerShell** - Windows automation

## üí° Pro Tips

1. **Master the basics** - Don't jump to advanced tools without understanding fundamentals
2. **Document everything** - Today's notes are tomorrow's writeups
3. **Practice deliberately** - Focus on weak areas, not just comfortable topics
4. **Build tools** - Writing your own scripts deepens understanding
5. **Stay legal** - Only practice on authorized targets (THM, HTB, your own labs)
